# Test Events Validation Guide

## Objective
Ensure all pixel events fire correctly before running ads.

---

## Validation Steps

1. Open Meta Events Manager  
2. Go to Test Events  
3. Enter website URL  
4. Perform user actions:

   - Visit page  
   - View product  
   - Submit form  

5. Confirm events appear in real time.

---

## Common Issues Checked

- Pixel not firing  
- Duplicate events  
- Wrong event parameters  
- Delayed firing  

---

## Success Criteria

✅ Events firing in real time  
✅ Correct event names  
✅ No duplicate warnings

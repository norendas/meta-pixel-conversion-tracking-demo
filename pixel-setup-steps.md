# Meta Pixel Setup Steps

## Step 1: Create Pixel

1. Go to Meta Events Manager  
2. Click "Connect Data Source"  
3. Select Web → Meta Pixel  
4. Enter Pixel Name  
5. Add website URL  

---

## Step 2: Install Base Code

Place the Meta Pixel base code inside the <head> section of the website.

---

## Step 3: Verify Installation

Use Meta Pixel Helper Chrome extension to confirm:

- Pixel firing  
- No duplicate events  
- Correct Pixel ID  

---

## Step 4: Configure Events

Set up standard events based on funnel stages.
